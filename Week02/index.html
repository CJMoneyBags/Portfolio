Notes

  • Creating Array Literals

    • const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];

  • Removing Values from Arrays

    • delete avenger[3];
    • This will remove the index of 3 so it will remove 'Hulk'

  • Destructuring Arrays

    • const [x,y] = [1,2];
    • This makes swaping values easy with:
      • [x,y] = [y,x];

    • Array Properties

      • length: shows the amount of elements
      • pop: avengers.pop(); this removes the last element
      • shift: avengers.shift(); this returns the first element
      • push: avengers.push('Thor'); this will append a new value
      • unshift: avengers.unshift('Captain America'); this adds item to the begining of the list
      • Merging arrays
        • concat: avengers = avengers.concat(['Hulk','Hawkeye', 'Black Widow']); this will add the list to the original list
